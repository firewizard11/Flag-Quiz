# Flag Quiz

## Description

Flag Quiz is a simple GUI quiz game written using Java Swing

## How to Play

```bash
java Game.java
```

## Reference

- <https://flagpedia.net/download>
- <https://datahub.io/core/country-list>

# Ideas
- Multiplayer
- Timer/Time limit
- Answer submit automatically
- No need for spaces
- Can enter acronyms instead of country name
